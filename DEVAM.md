## 8. UXP API (resmî `@adobe/premierepro` tiplerinden doğrulandı) + tuzaklar
- `const ppro = require("premierepro")`. Çoğu method **Promise → await**.
- **Transaction kalıbı:** `project.lockedAccess(() => project.executeTransaction((ca) => ca.addAction(action), "undo etiketi"))`. *(Blueprint'teki düz `executeTransaction`'dan farklı — `lockedAccess` ile sarılır, callback `compoundAction` alır, `ca.addAction(...)`.)*
- **Import (stereo):** `await project.importFiles(paths, true /*suppressUI*/, undefined /*bin*/, false /*stills*/)`.
- **Proje öğeleri:** `await project.getRootItem()` → `await root.getItems()`; `ppro.ClipProjectItem.cast(item)`; `await clip.getMediaFilePath()`.
- **Kaynak kırpma:** `clipProjectItem.createSetInOutPointsAction(inTT, outTT)` (yerleştirmeden ÖNCE → handle payı kalır).
- **Sequence:** `await project.createSequenceFromMedia(name, [clipItem])` → medya ayarlarıyla (1080p/24fps/stereo); preset dosyası gerekmez. (Alt.: `createSequenceWithPresetPath`.)
- **Yerleştirme:** `ppro.SequenceEditor.getEditor(seq).createOverwriteItemAction(item, timeTT, vIdx, aIdx)` — index **0=V1/A1** (offset değil). `createInsertProjectItemAction(..., limitShift)` alternatif.
- **Geçiş:** `await ppro.TransitionFactory.getVideoTransitionMatchNames()` → `await ppro.TransitionFactory.createVideoTransition(name)`; `const o=ppro.AddTransitionOptions(); o.setApplyToStart(bool); o.setDuration(TickTime); o.setTransitionAlignment(number); o.setForceSingleSided(bool);` → `trackItem.createAddVideoTransitionAction(vt, o)`.
- **TickTime:** `createWithSeconds(s)`, `createWithTicks("...")`, `createWithFrameAndFrameRate(f, fr)`, `TIME_ZERO`/`TIME_ONE_SECOND`; `.seconds`, `.ticksNumber`. **ticksPerSec = 254016000000.**
- **Manifest:** UXP `manifestVersion: 5`, `host.app: "premierepro"`, panel entrypoint. Build gerekmez (düz JS). UDT → Add Plugin → `panel/manifest.json` → Load. Premiere: **Window → UXP Plugins**. Developer Mode şart (Settings → Plugins).
